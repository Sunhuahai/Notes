### git知识
git add .将修改提交到暂存区
git commit 将暂存区的修改提交到当前分支

生成分支：git branch dev
切换分支：git checkout dev
生成并切换分支：git checkout -b dev
查看分支：git branch
合并到当前分支：git merge dev
删除分支: git branch -d dev
